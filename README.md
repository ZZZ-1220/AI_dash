# AI_dash

## ILCO GOODS DATA LAB

「SNS 시대의 일코 굿즈 마케팅」 보고서·발표자료·검수 데이터베이스를 바탕으로 만든 정적 데이터 대시보드입니다.

## 주요 기능

- Overview, Market & Cases, Price Intelligence, Pilot & KPI, Data Explorer의 5개 화면
- 93개 검수 레코드 검색·필터·정렬·페이지네이션
- 레코드별 출처, 검증 주의, 산출식, 검수 조치 상세 조회
- 필터 결과 CSV 다운로드
- 사례 상세 패널과 가격 범위 시각화
- 12주 파일럿 의사결정 상태 로컬 저장
- 데스크톱·태블릿·모바일 반응형 UI

## 실행

별도 빌드 도구나 패키지가 필요하지 않습니다.

    python3 -m http.server 4173

브라우저에서 http://localhost:4173 을 엽니다.

## 데이터 원칙

- 기준일: 2026-07-15
- `약`, `이상`, `돌파`, `전망` 표현을 원문 그대로 유지
- 굿즈 매출과 인접 게임 IP 성과를 분리
- C2C 가격은 실제 체결가가 아닌 게시 호가로 표시
- 이질적인 가격군의 통합 평균을 산출하지 않음

`main` 브랜치에 변경사항이 반영되면 GitHub Actions가 정적 파일을 GitHub Pages로 배포합니다.
